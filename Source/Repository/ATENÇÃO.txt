Cada um deve identificar as portas que está a usar para não haver conflito de portas.

Vamos dividir o código em três partes:
- Controlo de temperatura  VITOR
- Controlo de porta        JOEL
- Leitura de QR CODE       NUNO

Genérico: usar a variavél booleana "open"

Controlo de temperatura:
Se open = true não faz nada;
Se open = false, o sensor lê a temperatura e faz o controlo, 
a cima de x graus ativa a ventoinha e a baixo ativa o led vermelho

Controlo da porta:
Se open = false não faz nada;
Se open = true, abre a porta, espera x tempo e fecha automaticamente. Se durante o fecho
algum objeto for detetado, a porta volta a abrir e a esperar o respetivo tempo e assim
sucessivamente. Quando a porta estiver completamente fechada, o estado "open" deve mudar
para false.
